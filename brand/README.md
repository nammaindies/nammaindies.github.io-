# Brand assets

The mark is a curled tail — it reads as a dog's or a cat's tail, and as a
curled-up sleeping animal. One mark, used everywhere: logo, favicon, social
avatar, section ornament.

- `mark.svg` — the source mark (same path as the site's `favicon.svg` and
  inline logos). Scalable; use this wherever you can.
- `mark-1080-light.png` — 1080×1080 terracotta-on-paper. Social profile
  avatar (safe for circular crop).
- `mark-1080-dark.png` — 1080×1080 light-on-ink, for dark avatars.

The site itself renders the logo as inline SVG and `favicon.svg`, so the
pages don't load these PNGs — they're here for uploads (X, Instagram) and as
the versioned source of truth.

Colours: ink `#2b2620`, paper `#faf7f2`, terracotta `#a5502e` (dark-mode
terracotta `#e0895a`). Placeholder identity, pending a real designer.
